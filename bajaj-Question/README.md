# Bajaj Training - DSA Practice

Java solutions for Data Structures and Algorithms problems practiced during Bajaj Finserv training.

## Structure

```
day-wise/
├── Day-1/      Sliding Window & Two Pointers
├── Day-2/      Prefix Sum & Split Array
├── Day-3/      Binary Search + OOP (Inheritance, Overloading, Overriding)
├── Day-4/      Circular Arrays & Kadane's Algorithm
├── Day-5/      Strings & Hash Maps
├── Day-6/      Linked Lists (Add Two Numbers, KMP)
├── Day-7/      Miscellaneous
└── Daily/      Date-wise practice files (April 24, 25, 27, 28 | May 5)

problems/
├── arrays/         Two Sum, Rotate Array, Move Zeroes, Buy & Sell Stocks, Sort Colors, etc.
├── binary-search/  Search Insert, Find Peak, Koko Bananas, Ship Packages, Rotated Array, etc.
├── strings/        Valid Anagram, Valid Palindrome, Longest Substring, Repeated DNA, etc.
├── linked-lists/   Reverse LL, Cycle Detection, Merge Lists, Middle Node, Palindrome LL, etc.
├── trees/          Inorder/Preorder/Postorder, Level Order, Invert, Diameter, BST, etc.
├── graphs/         Number of Islands, Flood Fill, Course Schedule
├── stack/          Valid Parentheses, Generate Parentheses
└── dp/             House Robber
```

## Topics Covered

- Sliding Window, Two Pointer
- Prefix Sum
- Binary Search
- Arrays & Hashing
- Strings
- Linked Lists
- Binary Trees & BST
- Graphs (BFS/DFS)
- Stack & Backtracking
- Dynamic Programming
- OOP (Inheritance, Method Overloading, Overriding)
